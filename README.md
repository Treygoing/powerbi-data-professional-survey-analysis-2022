# powerbi-data-professional-survey-analysis-2022
Power BI dashboard analysing survey data from 630 data professionals to explore salaries, skills, and career satisfaction.
## Overview

This Power BI dashboard analyses survey responses from 630 data professionals to better understand roles, salaries, skills, and job satisfaction across the industry.

The goal of this project was to practise survey data cleaning, categorical analysis, and building comparative visualisations.

## Problem Statement

The data industry is growing rapidly, but questions remain around:

- Which roles earn the highest salaries?  
- What programming languages are most popular?  
- How satisfied are professionals with salary and work-life balance?  
- How difficult is it to break into data roles?  

This dashboard presents clear visual answers to these questions.

---

## Dataset

- Survey dataset with **630 respondents**  
- Includes:
  - Country  
  - Job Title  
  - Salary  
  - Programming Language  
  - Satisfaction Scores  
  - Age  

Survey responses were cleaned and standardised for consistent analysis.

---

## What I Did

- Cleaned inconsistent job titles and responses  
- Standardised salary data  
- Created measures for:
  - Average Salary by Job Title  
  - Satisfaction Scores  
  - Respondent Counts  
- Built visual comparisons using:
  - Bar charts  
  - Donut charts  
  - Gauge visuals  
  - KPI cards  
- Structured layout for intuitive comparison and insight discovery  

---

## Key Insights

- 630 total respondents  
- Average age: **29.87**  
- Data Scientists and Data Engineers report the highest salaries  
- Python is the most popular programming language  
- Most respondents find breaking into data neither easy nor very difficult  
- Work-life balance is rated higher than salary satisfaction
